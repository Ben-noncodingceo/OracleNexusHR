# 🚀 快速使用指南

## 第一步：启动服务器

### 方式一：自动启动（推荐）⭐

**Windows 用户**：
1. 双击运行 `start.bat` 文件
2. 等待依赖自动安装（首次运行）
3. **浏览器会自动打开！** 🚀
4. 看到启动成功提示

**Mac/Linux 用户**：
1. 打开终端
2. 进入项目目录
3. 运行命令：
   ```bash
   ./start.sh
   ```
4. **浏览器会自动打开！** 🚀

**特点**：
- ✅ 自动检查并安装依赖
- ✅ 自动启动服务器
- ✅ **自动打开浏览器**
- ✅ 无需手动输入网址！

### 方式二：使用启动器页面

如果脚本无法使用，可以：
1. 双击打开 `launcher.html` 文件
2. 按照页面提示启动服务器
3. 服务器启动后，点击"打开应用"按钮

**启动器特点**：
- 📱 友好的图形界面
- 🔍 自动检测服务器状态
- 📋 一键复制启动命令
- 💡 详细的分步指南

## 第二步：验证启动成功

浏览器打开后，检查页面顶部状态：
- ✅ 如果显示 **🟢 服务器运行正常**，说明一切正常
- ❌ 如果显示 **🔴 服务器未启动**，返回第一步重新启动

## 第三步：配置 API

### 推荐使用 DeepSeek（便宜、中文好）

1. 访问 https://platform.deepseek.com/
2. 注册账号并登录
3. 进入 API Keys 页面
4. 创建新的 API Key
5. 复制 API Key（格式：sk-xxxxxxxx）
6. 在网页的"AI 配置"部分：
   - 选择：DeepSeek
   - 粘贴 API Key
7. 点击 **🔍 测试 API 连接** 按钮
8. 看到 **✅ API 连接成功** 即可

### 备选使用 OpenAI

1. 访问 https://platform.openai.com/
2. 其他步骤同上
3. 在网页选择：OpenAI

## 第四步：开始分析

1. 填写个人信息：
   - 姓名：张三
   - 出生日期：选择公历日期
   - 出生时间：选择具体时间（24小时制）

2. 点击 **开始分析命理** 按钮

3. 等待 10-30 秒

4. 查看分析结果：
   - 📋 基本信息
   - 🎴 生辰八字（年柱、月柱、日柱、时柱）
   - ⭐ 星座与月相
   - 💫 命理建议
   - 🌙 星座运势建议
   - 🌕 月相能量指引

## 常见问题

### Q: 显示"服务器未启动"怎么办？
A: 请确保按照第一步启动了服务器。必须先启动服务器才能使用。

### Q: API 连接失败怎么办？
A:
1. 检查 API Key 是否正确（复制完整，包括 sk- 前缀）
2. 点击"测试 API 连接"查看具体错误
3. 检查账号是否有余额
4. 查看错误代码：
   - HTTP_401: API Key 无效
   - HTTP_429: 配额不足
   - NETWORK_ERROR: 网络连接问题

### Q: 下载日志失败怎么办？
A: 确保服务器正在运行（看顶部状态指示器是否为绿色）

### Q: 如何停止服务器？
A: 在运行 start.bat 或 start.sh 的终端/命令行窗口按 `Ctrl+C`

## 注意事项

⚠️ **重要**：
- 必须先启动服务器，再打开网页
- 不要直接双击打开 index.html 文件
- 正确方式：启动服务器 → 访问 http://localhost:3000
- 使用完毕后可以关闭终端窗口停止服务器
- API Key 请妥善保管，不要泄露给他人

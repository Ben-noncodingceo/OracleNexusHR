### ❌ 为什么不能用 GitHub Pages？

**GitHub Pages 只支持静态网站**（纯 HTML/CSS/JavaScript），不支持：
- ❌ Node.js 后端服务器
- ❌ Express API
- ❌ 服务器端运算

您的八字命理系统需要运行 Node.js 服务器，所以**无法部署到 GitHub Pages**（xxx.github.io）。

**但是**，通过 Vercel 部署，您会获得：
- ✅ 免费域名：`xxx.vercel.app`
- ✅ 免费 HTTPS 证书
- ✅ 全球 CDN 加速
- ✅ 自动持续部署

## 🆘 故障排除

### 问题：找不到 Vercel 项目

**解决**：
1. 确认已登录正确的 Vercel 账号
2. 检查是否已导入 GitHub 仓库
3. 如未导入，访问 https://vercel.com/new 导入

### 问题：部署失败

**解决**：
1. 检查 Vercel 部署日志
2. 确认 `vercel.json` 文件存在
3. 确认 `package.json` 包含正确的依赖

### 问题：API 调用失败

**解决**：
1. 检查 DeepSeek API Key 是否有效
2. 访问 https://platform.deepseek.com/ 验证
3. 检查账号余额是否充足

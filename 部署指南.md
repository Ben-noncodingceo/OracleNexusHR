# 🚀 免费部署指南

## 方案一：Vercel 部署（推荐⭐）

### 为什么选择 Vercel？

- ✅ **完全免费** - 个人项目永久免费
- ✅ **支持 Node.js** - 可以运行后端服务器
- ✅ **自动部署** - 推送到 GitHub 自动更新
- ✅ **免费域名** - 提供 `xxx.vercel.app` 域名
- ✅ **全球 CDN** - 访问速度快
- ✅ **免费 HTTPS** - 自动配置 SSL 证书
- ✅ **零配置** - 已准备好 `vercel.json` 配置文件

### 部署步骤

#### 1. 注册 Vercel 账号

访问 https://vercel.com/signup

选择 "Continue with GitHub" 使用 GitHub 账号登录

#### 2. 导入 GitHub 仓库

1. 登录后，点击 "Add New..." → "Project"
2. 选择 "Import Git Repository"
3. 找到您的仓库 `OracleNexusHR`
4. 点击 "Import"

#### 3. 配置项目（通常自动检测）

Vercel 会自动检测到这是一个 Node.js 项目。

**重要设置**：
- Framework Preset: `Other`
- Build Command: 留空或 `npm install`
- Output Directory: 留空
- Install Command: `npm install`

#### 4. 部署

点击 "Deploy" 按钮，等待 1-2 分钟

部署成功后，您会获得一个域名，例如：
```
https://oracle-nexus-hr-xxx.vercel.app
```

#### 5. 访问您的应用

打开分配的域名，就可以使用了！

### 自动更新

以后每次您推送代码到 GitHub，Vercel 会**自动重新部署**，无需手动操作。

## 方案二：Render 部署（备选）

Render 也是一个优秀的免费平台。

### 部署步骤

#### 1. 注册账号

访问 https://render.com/

使用 GitHub 账号注册

#### 2. 创建 Web Service

1. 点击 "New +" → "Web Service"
2. 连接您的 GitHub 仓库 `OracleNexusHR`
3. 配置：
   - **Name**: `bazi-fortune-calculator`
   - **Environment**: `Node`
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Instance Type**: `Free`

#### 3. 部署

点击 "Create Web Service"

等待部署完成（约 3-5 分钟）

您会获得一个域名：
```
https://bazi-fortune-calculator-xxx.onrender.com
```

### 注意事项

- 免费版会在无访问时休眠
- 首次访问可能需要 30-60 秒启动
- 推荐使用 Vercel（更快）

## ⚠️ 为什么不能用 GitHub Pages？

**GitHub Pages 只支持静态网站**（纯 HTML/CSS/JavaScript），不支持：
- ❌ Node.js 服务器
- ❌ Express 后端
- ❌ API 端点
- ❌ 服务器端运算

您的应用需要运行 Node.js 服务器，所以必须使用支持后端的平台。

## 🎯 推荐部署流程

### 快速部署（5分钟）

1. **推送代码到 GitHub**（如果还没推送）
   ```bash
   git add .
   git commit -m "准备部署到 Vercel"
   git push
   ```

2. **访问 Vercel**
   - 打开 https://vercel.com/signup
   - 使用 GitHub 登录

3. **导入项目**
   - 点击 "Add New..." → "Project"
   - 选择 `OracleNexusHR` 仓库
   - 点击 "Import"

4. **部署**
   - 点击 "Deploy"
   - 等待完成

5. **完成！**
   - 复制分配的域名
   - 在浏览器打开
   - 开始使用！

## 🌐 自定义域名（可选）

### Vercel 自定义域名（免费）

1. 在 Vercel 项目设置中
2. 进入 "Domains"
3. 添加您的域名
4. 按照提示配置 DNS

支持：
- 您自己的域名（需要购买）
- 免费的 `.vercel.app` 子域名
